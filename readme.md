# Hello World HTML Page

## Project Description

This is my first project, a simple "Hello, World!" web page built using basic HTML5. The goal of this project was to learn the fundamental structure of an HTML document, including the use of `<!DOCTYPE html>`, `<html>`, `<head>`, `<title>`, and `<body>` tags.

The project consists of a single file: `index.html`.

## How to Run
I used live server to run the html file. 
and use vercel for deployment.
